# tomodachi

This is an application I'm making to help me learn
nihongo.

It'll be a "Who Wants to be a Millionaire?" style quiz 
with four options.

First, it'll be for learning the hiragana, the katakana
and the Kanji.

It's designed to be used on the epaper-hat.
